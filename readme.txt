En la carpeta ‘root’ existe el directorio ‘src’. En el mismo se encuentra el archivo syntax.l 
en donde está el código del programa. En la raíz del directorio existe el archivo test_file.sh
el cual contiene varias pruebas del programa. Al ejecutarse compila el programa y 
automáticamente corre varios tests para probar el correcto funcionamiento del programa. 
Para compilarlo, se utiliza un archivo makefile y los tests más importantes se encuentran 
en el directorio ‘examples’.

Dentro del directorio ‘examples’, se encuentran archivos con cadenas genómicas. 
Enre los mismos se pueden encontrar las cadenas provistas por la cátedra, así como 
también otras cadenas obtenidas de una fuente de internet .

Al ejecutar el archivo test_file.sh, se obtiene en el stdout las cadenas de 
aminoácidos traducidas a partir de las cadenas genómicas.
